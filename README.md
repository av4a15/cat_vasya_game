# cat_vasya_game
  Модуль 1 
  
Класс Unit - абстрактный. Есть 6 характеристик. Есть три абстрактных метода.
Класс Character - наследник Unit. Реализует методы.
Класс Monster - наследник Unit. Свои формулы.
 
  Модуль 2
  
Один класс Character вместо трёх.
Внутри методов проверяется character_class. Для каждого класса своя формула.
В конструкторе сохраняются здоровье, урон и защита.
Если класс неизвестный - ошибка.
  
  Модуль 3
  
Класс Spell - абстрактный. Есть название, урон, стоимость. Метод cast().
У персонажей есть заклинания. Их три штуки. У каждого свой урон и стоимость.
Так же у персонажей есть мана. Её формула зависит от класса. У монстров нет маны.
В класс Unit добавлены список заклинаний, мана, методы для добавления и применения заклинаний.
При применении проверяется, хватает ли маны.
У персонажей есть мана, её формула зависит от класса. У монстров нет маны.
